# meu plano para o projeto

Projeto desenvolvido para o projeto Agrinho
Plano de Projeto: Agrinho

1. Objetivo do Projeto
Desenvolver nos estudantes consciência ambiental, valores sociais e cidadania.
Incentivar a participação ativa em ações de sustentabilidade e cuidado com a comunidade.
Promover integração entre escola, família e comunidade.

2. Público-alvo
Alunos do ensino fundamental (1º ao 9º ano).
Professores e equipe escolar.
Comunidade local (opcional).

3. Duração do Projeto
Planejamento: 2 semanas
Execução: 3 a 4 meses (podendo ser adaptado de acordo com o calendário escolar)
Avaliação: 1 a 2 semanas
